# 🏨 Smart Hostel Management App

A Flutter-based application to manage hostel activities efficiently including attendance, mess management, and student feedback.

---

## 🚀 Getting Started

This project is developed using Flutter and Firebase to digitize hostel operations.

### 📌 Features

* Student Dashboard
* Mess Management System
* Attendance Tracking
* Feedback System
* Special Menu Updates
* Resource Estimation (Rice, Curry, etc.)

---

## 🛠️ Technologies Used

* Flutter
* Firebase
* Cloud Firestore
* Firebase Authentication

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/vengalarakshitha7-lang/hostel-management-app.git
```

2. Navigate to project folder:

```
cd hostel-management-app
```

3. Get dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```

---

## 📂 Project Structure

* `lib/` → Main application code
* `android/` → Android configuration
* `ios/` → iOS configuration
* `assets/` → Resources
